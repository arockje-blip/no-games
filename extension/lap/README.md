# Lap Guard

Lap Guard is a Chrome/Edge Manifest V3 extension that blocks a curated list of gaming and game-download domains, with a password-protected pause flow and a keyboard shortcut to open the dashboard.

## Quick start

1. Open `edge://extensions` or `chrome://extensions`.
2. Turn on Developer mode.
3. Load the `lap` folder as an unpacked extension.
4. Open the extension popup or press `Ctrl+Shift+L`.
5. Sign in, then choose a pause window of 2, 5, 10, 15, 25, 35, 50, or 60 minutes.

## What it blocks

The default list covers common gaming stores, launchers, and browser game sites such as Roblox, Steam, Epic Games, Battle.net, Riot, EA, Ubisoft, Nintendo, PlayStation, Xbox, GOG, itch.io, and several browser game portals.

## Notes

- This is browser-level blocking, not full device-wide DNS control.
- If you want device-wide blocking, mirror the same domain list into a hosts-file or DNS filter solution.
- The block list is editable in the Options page after unlocking.
